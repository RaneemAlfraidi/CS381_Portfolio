### Repository Structure
The project is organized into a modular directory structure for clarity and maintenance:

* **/** : Primary HTML documents (`index`, `profile`, `contact`).
* **/css** : Contains `style.css` for Global styles and layout.
* **/docs** : Technical documentation (Design PDF & Wireframe).
* **/images** : Media assets and profile photography.
